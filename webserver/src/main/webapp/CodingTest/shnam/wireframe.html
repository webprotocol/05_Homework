01장 HTML 기본 다지기

오리 날다

이 장에서 꼭 외워야 할 태그들

<!doctype html> 웹 브라우저에게 HTML 형식에 맞춰 문서를 해석하라고 알려주는 태그
<html> 웹문서의 시작과 끝을 나타내는 태그
<head> 브라우저창에는 보이지 않지만 중요한 문서 정보를 삽입하는 영역 태그
<body> 웹 문서의 실제 내용이 들어가는 영역 태그
<meta> 문자 세트나 키워드 등의 정보를 지정하는 태그
<title> 웹브라우저 창의 제목 표시줄에 표시될 문서 제목을 나타내는 태그
<br> 웹 문서에서 줄 바꾸기 태그
<img> 웹 문서에 이미지를 넣는 태그

02장 HTML5와 시맨틱 태그

이 장에서 꼭 외워야 할 태그들

<header> 사이트나 특정 부분의 머리말(제목 부분)태그
<nav> 문서나 사이트를 연결하는 내비게이션 태그
<hn> 크기별로 텍스트 제목을 붙이는 태그
<article> 실제 콘텐츠 영역을 나타내는 태그
<aside> 본문 이외의 사이드 바 영역 태그
<footer> 사이트 제작자 연락 정보나 저작권 정보가 포함되는 푸터 태그

03장 텍스트와 하이퍼링크 관련 태그들

이 장에서 꼭 외워야 할 태그들

<p> 웹 문서에서 텍스트 단락을 만드는 태그
<blockquote> 웹 문서에 인용문을 넣는 태그
<mark> 중요한 일부 텍스트 부분에 형광펜으로 그은 것처럼 표시하는 태그
<strong>,<b> 일부 텍스트를 진하게 표시하는 태그
<em>,<i> 일부 텍스트 이탤릭체로 표시하는 태그
<ul>,<ol>,<li> 순서 없는 목록, 순서 목록을 만드는 태그
<a> 문서와 사이트를 옮겨 다니는 링크를 만드는 태그
<table>,<tr>,<td>,<th> 표를 만들 때 사용하는 태그

04장 밀티미디어를 다루는 태그들

이 장에서 꼭 외워야 할 태그들

<img> 웹 문서에 이미지를 넣는 태그
<figure>,<figcaption> 미디어 파일에 캡션을 붙이는 태그
<embed> HTML4와 HTML5에서 미디어 파일을 재생하는 태그
<video> 플러그인 프로그램 없이 비디오를 재생하는 태그
<audio> 플러그인 프로그램 없이 오디오를 재생하는 태그
<source> 미디어 파일의 경로를 지정하는 태그

04장 밀티미디어를 다루는 태그들

이 장에서 꼭 외워야 할 태그들

<form> 폼 전체를 감싸는 태그
<fieldset>,<legend> 폼 요소를 그룹으로 묶고 제목을 붙이는 태그
<label> 폼 요소에 캡션을 붙이는 태그
<input> 사용자가 입력할 수 있도록 여러 요소를 삽입하는 태그
<select>,<option> 드롭다운 목록을 넣는 태그
<textarea> 여러 줄의 텍스트를 입력하는 텍스트 영역을 넣는 태그
<button> 버튼을 넣는 태그

06장 스타일 시트를 정의하는 태그

이 장에서 꼭 외워야 할 태그들

<style> 스타일 시트를 정의하는 태그
.class 특정 부분에 스타일을 적용하는 클래스 선택자
#id 특정 부분에 스타일을 적용하는 id 선택자
요소1 요소2 모든 하위 요소에 스타일을 적용하는 하위 선택자
요소1 < 요소2 자식 요소에만 스타일을 적용하는 자식 선택자
요소1 + 요소2 첫 번째 동생 요소에 스타일을 적용하는 인접 형제 선택자
요소1 ~ 요소2 모든 형제 요소에 스타일을 적용하는 형제 선택자

07장 텍스트 관련 스타일

이 장에서 꼭 외워야 할 속성들

font-size 글자 크기를 지정하는 속성
fon-wieght 글자 굵기를 지정하는 속성
text-align 텍스트 정렬 방법을 지정하는 속성
text-shadow 텍슽에 그림자를 지정하는 속성
line-height 단락 안에서 줄간격을 조절하는 속성
list-style 불릿 유형과 위치를 조절하는 속성, 필요할 경우 이미지를 대신 사용할 수 있음

color 글자색을 바꾸는 속성

background-image 배경 이미지 파일 경로를 지정하는 속성
background-repeat 배경 이미지 반복 여부를 지정하는 속성
background-position 배경 이미지 위치를 지정하는 속성

09레이아웃을 위한 스타일

이 장에서 꼭 외워야 할 속성들

width,height 박스 모델의 크기를 지정하는 속성
margin 박스 모델 주변 여백을 설정하는 속성
padding 콘텐츠 영역과 테두리 사이의 여백을 설정하는 속성
border 테두리 두께와 색, 스타일을 지정하는 속성
border-radius 테두리 모서리를 둥글게 처리하는 속성
box-shadow 박스 모델에 그림자를 추가하는 속성
position 웹 문서에서 특정 요소를 원하는 위치에 배치하는 속성

10장 변형을 위한 스타일

transfrom 웹 요소를 변형하는 속성
translate 웹 요소 이동시키기

animation CSS3에서 애니메이션을 만드는 속성

11장 반응형 웹 디자인과 미디어 쿼리

이 장에서 꼭 외워야 할 것들

@media 미디어 쿼리를 적용할 미디어 종류를 지정
min-width, min-height [미디어 쿼리 조건에서]최소 너비, 최소 높이
device-width, device-height [미디어 쿼리 조건에서]단말기 너비와 높이
0rientation [미디어 쿼리 조건에서]단말기의 회전 방향
aspect-ratio [미디어 쿼리 조건에서]화면 비율
